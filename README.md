## Description
This project is a simple Python keylogger that captures keyboard input using the pynput library and saves the recorded keystrokes into a text file for educational purposes.

## Features
- Logs all key presses  
- Supports letters, numbers, and special keys  
- Saves keystrokes to a text file  
- Stops logging when ESC is pressed  
- Works in the background  

## Requirements
- Python 3.x  
- pynput library  

Install pynput using:
pip install pynput

## How to Run
1. Open Command Prompt / PowerShell  
2. Navigate to the project folder  
3. Run the program:

   python Task4.py

4. Type anything on the keyboard  
5. Press ESC to stop logging  

## Output
All keystrokes are saved in:
keystrokes.txt
